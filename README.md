# AI Нормконтролер

Система автоматической проверки инженерных документов на соответствие ГОСТ 2.105-95 с использованием LLM.

## Возможности
- 🔍 Семантический поиск релевантных правил ГОСТ
- 🤖 Анализ текста через LLM (OpenRouter)
- 📄 Генерация подробных отчетов о нарушениях
- 🐳 Полная поддержка Docker

## Быстрый старт

### Локальный запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Постройте индекс: `python src/index_builder.py`
3. Запустите сервер: `uvicorn backend.app:app --reload`
4. Откройте http://localhost:8000/docs

### Запуск в Docker
```bash
docker-compose up --build
